# Simple Text (beta)
A very simple text and code editor written in Vala.

Compiling: make simple-text

Requires: vala, gtk+-3.0, gtksourceview-3.0

#### Currently working:
* Open files
* Save files
* Multiple tabs
* Toggle line numbers
* Regular text editor stuff

#### Not working:
* Syntax highlighting
* Style schemes
* Text/Code completion
* Brace completion
* Re-open last tab
* To be defined...

#### Known bugs
* None (yet)
